📁 Hearts of Iron IV 本体目录结构说明

steamapps > common > Hearts of Iron IV
  • assets
    基础资源文件，包含游戏核心资产和依赖文件
    • browser
      内置浏览器相关文件
  • browser
    内置浏览器相关文件
  • cef
    Chromium Embedded Framework 文件
  • common
    常用文件，包含大量按钮制作和通用脚本文件
  • country_metadata
    国家元数据存储
  • crash_reporter
    崩溃报告工具
  • clc
    钢线棒/CNV DLC文件，包含扩展内容
  • clc_metadata
    DLC元数据文件
  • documentation
    钢线棒/CNV文档文件，包含大量条件、效果、修正等实用词条
  • EmptySteamDepot
    Steam空仓库占位文件
  • events
    事件文件夹，新闻及普通事件都在这里面定义
  • gfx
    美术资源文件夹，包含所有游戏图形资源
  • history
    历史文件夹，可定义国家、地块、军队、通用将领顾问
  • integrated_clc
    集成DLC内容
  • interface
    页面文件夹，可定义图片图标、GUI前端、鸣谢名单修改等
  • localisation
    本地化文件夹，包含游戏文本的多语言翻译
  • map
    地图文件夹，包含游戏地图数据和定义
  • music
    音乐包文件夹，包含游戏背景音乐
  • pdx_browser
    Paradox浏览器组件
  • pdx_launcher
    游戏启动器文件
  • pdx_online_assets
    在线资源文件
  • portraits
    随机大众脸肖像定义，包含领导人肖像
  • previewer_assets
    预览资源文件
  • script
    游戏脚本文件
  • sound
    音效文件夹，包含游戏音效和语音
  • steam_settings
    Steam平台设置文件
  • tests
    测试相关文件
  • tools
    工具文件夹，包括一些钢网制作组常用的工具
  • tutorial
    教程文件
  • tweakergui_assets
    调试GUI资源
  • wiki
    游戏维基文档
